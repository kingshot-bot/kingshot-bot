## FAQ

### Is Kingshot Bot safe to use?
Kingshot Bot uses direct server communication without modifying game files. While no bot is 100% undetectable, we implement human-like delays and behavior patterns to minimize risk.

### Do I need an emulator?
**No!** Kingshot Bot connects directly to Kingshot servers. No BlueStacks, LDPlayer, or any Android emulator required.

### Can I run multiple accounts?
Yes! Depending on your plan:
- Free: 1 account
- Basic: 2 accounts
- Full: 5 accounts

All accounts can run simultaneously on the same PC.

### What happens when the free 2 hours run out?
The bot pauses until the next day (midnight UTC reset). Your game account stays logged in, and you can continue playing manually.

### Does it work on Mac/Linux?
Currently Windows only. Mac and Linux versions are on our roadmap.

### How do updates work?
Kingshot Bot auto-updates in the background. You'll always have the latest version with new features and game compatibility fixes.

### Can I get banned?
As with any automation tool, there's inherent risk. We minimize this with human-like behavior patterns, but use at your own discretion.

### Is my password safe?
Yes. Credentials are encrypted and stored locally on your device. We never transmit or store your game password on our servers.
